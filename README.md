# ImobFlow API

Projeto backend desenvolvido com foco em aprendizado e evolução em engenharia de software moderna.

## Objetivo

Praticar e evoluir conhecimentos em:

- Java
- Spring Boot
- Arquitetura Hexagonal
- Clean Architecture
- DDD
- SQL
- Docker
- Testes automatizados
- TDD
- Git/GitHub
- CI/CD
- Observabilidade
- Mensageria
- Microsserviços
- AWS futuramente

## Tecnologias

- Java 21
- Spring Boot
- Gradle
- MySQL
- Docker
- JUnit
- Mockito
- Insomnia
- Git/GitHub

## Arquitetura

O projeto será desenvolvido inicialmente como um monólito modular, evoluindo gradualmente para conceitos mais avançados de engenharia de software e microsserviços.
